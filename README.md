# Frontend Testing Guide — Jest & React Testing Library

An interactive, single-page guide that teaches frontend testing from basics to advanced concepts using Jest and React Testing Library.

## Features

* Clear explanations of testing fundamentals
* Unit, Integration, and E2E testing breakdown
* Practical code examples with Jest & RTL
* Async testing, mocks, and user interactions
* Built-in quiz for self-check
* Clean, modern, interactive UI

## Usage

Open the `index.html` file in your browser — no setup required.

## Tech Stack

* HTML, CSS, JavaScript
* Jest
* React Testing Library

## What You’ll Learn

* Why testing matters
* Testing types and strategy (testing pyramid)
* Writing tests with Jest
* Using matchers and mocks
* Testing React components with RTL
* Handling async logic and user events
